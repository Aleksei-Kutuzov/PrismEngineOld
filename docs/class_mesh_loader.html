◆ LoadOBJ()

PRISM_Mesh MeshLoader::LoadOBJ ( const char * filename,
AbstractCamera3D camera )
static

Загружает 3D-модель из файла в формате OBJ.

Формат OBJ поддерживает только геометрию (вершины, нормали, текстурные координаты). Материалы (MTL) не обрабатываются этим методом, и информация о текстурах игнорируется. Так-же в данный момент поддерживается только информация о вершинах, треугольниках и нормалях вершин. Если файл не найден или имеет неверный формат, метод может вернуть пустую или некорректную сетку. Если вы экспортируете из blender поставьте галочки на Triangulated Mesh (чтобы сетка загрузилась корректно) и Normals (если нужно освещение по Фонгу и похожие системы освещенияя)

Аргументы
filenameПуть к файлу OBJ для загрузки.
cameraКамера, которая будет использоваться для преобразования модели.
Возвращает
Объект типа PRISM_Mesh, содержащий данные загруженной модели.
